# SVN Compare Tool

一个基于 Electron + Vue 3 + Vite 的 SVN 文件对比工具，用于快速查看和同步两个目录之间的 SVN 变更。

## 功能特性

- **SVN 日志查询**: 根据天数、作者、日志内容筛选 SVN 提交记录
- **双目录对比**: 同时查看两个目录的文件变更情况
- **差异可视化**: 使用 Monaco Editor 展示文件差异，支持语法高亮
- **差异导航**: 快速跳转到上一个/下一个差异位置
- **差异同步**: 支持单个差异同步或全部同步到目标文件
- **文件保存**: 直接保存修改后的文件
- **历史记录**: 自动保存查询历史，方便快速切换

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **构建工具**: Vite 5
- **桌面框架**: Electron 28
- **UI 组件**: Element Plus
- **代码编辑器**: Monaco Editor
- **编码处理**: iconv-lite (支持中文 GBK/CP936)

## 安装

```bash
# 安装依赖
npm install

# 开发模式运行
npm run electron:dev

# 构建生产版本
npm run electron:build
```

## 使用说明

1. **选择目录**: 分别选择目录 A（源目录）和目录 B（目标目录）
2. **设置筛选条件**:
   - 天数: 查询最近 N 天的提交记录
   - 作者: 按提交作者筛选（可选）
   - 日志: 按提交日志内容模糊匹配（可选）
3. **查询 SVN**: 点击"查询SVN"按钮获取变更文件列表
4. **查看差异**: 点击文件查看两个目录中该文件的差异
5. **同步差异**:
   - 使用"同步当前差异"将单个差异从左边同步到右边
   - 使用"全部同步"将左边内容完全覆盖到右边
6. **保存文件**: 点击"保存"将修改写入目标文件

## 文件操作标识

- **A** (绿色): 新增文件
- **M** (橙色): 修改文件
- **D** (红色): 删除文件

## 系统要求

- Windows 操作系统
- 已安装 SVN 命令行工具
- Node.js 18+
